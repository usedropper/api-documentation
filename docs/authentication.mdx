# Authentication

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';
import useBaseUrl from '@docusaurus/useBaseUrl';


You have to authenticate all API calls by including your API key in the authorization header of every request you make to the Dropper API.

:::caution IMPORTANT
Secure your API key.
Do not commit your API keys to git, or use them plainly in client-side code.
:::

<br/>

### How can I get my API key?


To get and manage your API Key, log into the [Dropper dashboard](https://dashboard.usedropper.com/).
Once logged in, navigate to "API Settings" to obtain your API key.

<br/>
<br/>

### Authorization format

Authorization headers should be in the following format: 


export const endpoint = 'https://api.usedropper.com/v1/graphql'

```jsx
const { GraphQLClient } = require("graphql-request");

const client = new GraphQLClient("https://api.usedropper.com/v1/graphql", {
  headers: {
    content-type: "application/json",
    // highlight-next-line
    x-dropper-api-key: process.env.YOUR_API_KEY
  },
});
```

<TabItem value="curl">

```bash
curl -X POST https://api.usedropper.com/v1/graphql \
-H "content-type: application/json" \
# highlight-next-line
-H "x-dropper-api-key: <YOUR_API_KEY>"
```
